# AI Code Optimizer

An AI-assisted code optimization tool that analyzes code and suggests improvements using LLMs.

- Analyze code for inefficiencies
- Suggest performance improvements
- Improve readability and structure
- Supports C++, Python, and general programming tasks

##  How It Works
1. User inputs code
2. The system sends structured prompts to an LLM
3. The model returns optimization suggestions
4. Results are displayed in a CLI interface
